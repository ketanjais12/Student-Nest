# StudentNest 🏠

StudentNest is a comprehensive student housing management platform designed to bridge the gap between students looking for comfortable, affordable accommodation and property owners managing hostels or PGs. Our goal is to make the relocation process simple, secure, and transparent.

## 🚀 Key Features

### For Students
*   **Verified Listings:** Search for hostels with real, verified photos and honest reviews.
*   **Simplified Booking:** Request a room directly through the platform with just a few clicks.
*   **Secure & Transparent:** Clear pricing, no hidden broker fees, and safe digital interaction.

### For Property Owners
*   **Dashboard Insights:** Track revenue, occupancy rates, and booking trends in real-time.
*   **Easy Management:** Manage multiple properties, update room availability, and adjust pricing instantly.
*   **Verified Requests:** Receive inquiries only from registered students, reducing spam and managing expectations effectively.

## 🛠 Tech Stack

**Frontend:**
*   React.js
*   Tailwind CSS
*   Redux Toolkit (State Management)
*   React Router (Routing)
*   Lucide React (Icons)
*   React Hot Toast (Notifications)

**Backend:**
*   Node.js & Express.js
*   MongoDB (Database)
*   JWT Authentication
